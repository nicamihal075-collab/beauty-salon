# Пользовательские сценарии

## Сценарий 1: Первичная запись на стрижку (Гость -> Пользователь)
1. Гость заходит на Главную страницу.
2. Переходит в раздел "Услуги".
3. Находит услугу "Мужская стрижка", нажимает "Записаться".
4. Система просит авторизоваться. Гость переходит на страницу регистрации.
5. Заполняет данные, нажимает "Зарегистрироваться".
6. После регистрации попадает на страницу записи, выбирает мастера и время.
7. Подтверждает запись.
8. **Результат:** Запись появляется в личном кабинете, администратор видит новую заявку.

## Сценарий 2: Проверка своих визитов (Пользователь)
1. Пользователь авторизуется в системе.
2. Переходит в "Личный кабинет".
3. Открывает вкладку "Мои записи".
4. Видит статус своей записи (например, "Подтверждено").
5. **Результат:** Пользователь информирован о статусе записи.

## Сценарий 3: Обработка новой заявки (Администратор)
1. Администратор заходит в панель управления (Админку).
2. Видит список новых записей, выделенных цветом.
3. Нажимает на запись клиента для просмотра деталей.
4. Проверяет график мастера и нажимает кнопку "Подтвердить".
5. **Результат:** Статус записи меняется, клиент видит обновление в своем кабинете.
