# Навигация и пользовательские маршруты

## 1. Карта навигации (Текстовая)
* **Шапка сайта (Header):** Логотип -> Главная | Услуги | Мастера | Контакты | [Войти / Профиль]
* **Подвал сайта (Footer):** Ссылки на соцсети, контакты, политика конфиденциальности.

## 2. Основные маршруты (User Flow)

### Маршрут гостя (Цель: Регистрация)
Главная страница → Просмотр услуг → Кнопка "Записаться" → Страница авторизации → Ссылка "Нет аккаунта?" → Регистрация → Личный кабинет.

### Маршрут пользователя (Цель: Запись на услугу)
Личный кабинет → Кнопка "Новая запись" → Выбор услуги → Выбор мастера → Выбор даты/времени → Подтверждение → Страница "Мои записи".

### Маршрут администратора (Цель: Обработка записи)
Вход в админ-панель → Список записей → Клик по новой заявке → Изменение статуса на "Подтверждена" → Сохранение.
