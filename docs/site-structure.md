# Структура сайта "BeautySpace"

## 1. Публичная часть (Доступна всем)
* **Главная страница (`index.html`)** — общая информация, акции, преимущества.
* **Каталог услуг (`services.html`)** — список категорий и услуг с ценами.
* **Страница мастеров (`masters.html`)** — список специалистов с их квалификацией.
* **О нас / Контакты (`contacts.html`)** — адрес, карта, режим работы.
* **Авторизация (`login.html`)** — форма входа.
* **Регистрация (`register.html`)** — форма создания аккаунта.

## 2. Личный кабинет пользователя (Для клиентов)
* **Профиль (`profile.html`)** — личные данные и настройки.
* **Мои записи (`my-bookings.html`)** — история и текущий статус заявок.
* **Форма онлайн-записи (`booking.html`)** — пошаговый выбор услуги, мастера и времени.

## 3. Административная панель (Для админа)
* **Дашборд (`admin/index.html`)** — общая статистика и уведомления о новых записях.
* **Управление записями (`admin/bookings.html`)** — просмотр всех заявок и смена их статусов.
* **Управление услугами (`admin/services.html`)** — добавление/редактирование прайса.
* **Управление мастерами (`admin/masters.html`)** — редактирование списка сотрудников.
